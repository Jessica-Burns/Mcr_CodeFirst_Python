Git/GitHub terms

  • Repository [repo]: a project where all of your files are, online or in your laptop
  • Commit: saving a given version of file(s)
  • Fork: make a copy of someone's else repository

Git/GitHub terms

  • Clone: copy a repo to your local computer
  • Pull request: request to add your changes into a forked repo
  • Merge: incorporating new changes from a PR/branch
